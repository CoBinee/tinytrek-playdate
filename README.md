# TINY TREK
**TINY TREK** は SF テレビドラマ「宇宙大作戦」を題材にしたシミュレーションゲームで、オリジナルである **STAR TREK** を、8bits コンピュータ向けに開発された BASIC 言語、Tiny BASIC 上で遊ぶために Tiny BASIC の開発者自らが移植したものです。

## 操作
- `クランク`<br>コマンドの選択、方向の設定
- `→`, `A`<br>決定
- `←`, `B`<br>キャンセル

## 開発
- Playdate SDK 2.0.0
- 美咲フォント
- Final Frontier Old Style Font
